Cara menambah artikel baru

Situs ini tidak memakai CMS — setiap artikel adalah satu berkas HTML biasa, supaya bisa langsung di-hosting di GitHub Pages tanpa proses build.

  1. Salin berkas articles/template-artikel.html, ganti nama sesuai judul artikel (mis. articles/nama-artikel-baru.html).
  2. Ubah judul, tanggal, dan isi tulisan di dalam berkas tersebut.
  3. Tambahkan satu kartu baru di halaman ini (articles.html) yang menaut ke berkas artikel tersebut.
  4. Commit dan push ke repository GitHub Anda — artikel otomatis tayang di GitHub Pages.
